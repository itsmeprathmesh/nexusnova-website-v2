# NexusNova Frontend Upgrade Notes

This version upgrades the public frontend from black/blue tech styling to a warmer luxury color system:

- Deep plum background instead of plain black
- Gold, rose, violet, and cyan accents
- Animated gradient mesh background
- Subtle mouse glow interaction
- 3D tilt cards using Framer Motion
- Floating hero dashboard mockup
- Stronger homepage copy and more sections
- Richer industries, services, case studies, testimonials, pricing, FAQ, and CTA content
- Improved contact page with premium cards
- Updated portfolio and blog pages with warmer visual system

Run after download:

```bash
npm install
npm run dev
```

Then check:

```bash
npm run type-check
npm run build
```

Note: node_modules are intentionally not included in the ZIP.
